Ikhtisar Design System

Ikhtisar Design System AOS

Selamat datang di spesifikasi visual resmi untuk AOS Online. Design system ini dibangun menggunakan basis CSS Variables modern, tipografi fungsional, dan visual terarah untuk memberikan user experience (UX) tingkat korporat yang kokoh, rapi, dan responsif.

Prinsip Desain Kami

Design system ini berpedoman pada tiga filosofi utama:

  • Kepercayaan Korporat (Corporate Trust): Didominasi oleh Royal Blue dan Ocean Blue untuk mewakili profesionalitas sistem organisasi.
  • Kesederhanaan Alur (Visual Guidance): Kontras tajam, label terstruktur, status indikator tegas, serta ilustrasi bernilai fungsional.
  • Responsif & Ringan: Dibangun murni menggunakan HTML5 semantik, CSS variabel vanilla, dan Javascript interaktif tanpa membebani browser.
Cara Penggunaan

Telusuri menu di panel samping untuk melihat token warna, ukuran spasi, aturan huruf, dan daftar komponen interface. Di setiap komponen, Anda dapat menggunakan tombol Copy HTML atau Copy CSS untuk langsung menempelkan markup ke dalam kode prototype Anda.

Ikonografi

Katalog SVG icon resmi yang terintegrasi di sidebar, panel worksheet, dan dialog box AOS Online. Semua ikon dirancang menggunakan style stroke minimalis.

Sumber Library Ikon
Lucide Icons
Library ikon SVG open-source berbasis stroke minimalis dengan ukuran kanvas standar 24x24px. Semua ikon di AOS Online digunakan sebagai inline SVG sehingga tidak memerlukan CDN atau koneksi internet.
Spesifikasi Teknis
viewBox 0 0 24 24
Fill none
Stroke currentColor
Stroke Width 2
Cara Penggunaan
① Inline SVG (Dipakai AOS)
Copy path langsung ke HTML, tidak butuh CDN:
<svg viewBox="0 0 24 24" fill="none"
  stroke="currentColor" stroke-width="2">
  <!-- path dari lucide.dev -->
</svg>
② CDN Script Tag
Tambah script lalu gunakan tag <i data-lucide>:
<script src="https://unpkg.com/
  lucide@latest/dist/umd/
  lucide.min.js"></script>

<i data-lucide="home"></i>
<script>lucide.createIcons();</script>
③ Browse Semua Ikon
Temukan 1500+ ikon di website resminya dan copy SVG path yang dibutuhkan:
lucide.dev/icons
